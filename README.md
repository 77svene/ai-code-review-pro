# ai-code-review-pro

> **Instant AI-powered code review, security audit, and refactoring assistant for modern development workflows.**

`ai-code-review-pro` is a lightweight, privacy-first tool designed to catch bugs, security vulnerabilities, and code style issues before you push to production. Whether you are building side projects or managing enterprise codebases, get professional-grade analysis in seconds.

---

## ✨ Features

*   **🔍 Deep Static Analysis:** Identifies logic errors, race conditions, and security flaws.
*   **🛡️ Security First:** Scans for common vulnerabilities (OWASP Top 10, XSS, SQLi).
*   **⚡ Fast & Local:** Runs efficiently in your environment with optional cloud processing.
*   **📝 Actionable Suggestions:** Provides concrete code snippets to fix issues immediately.

## 🛠️ Getting Started

### Installation

```bash
# Clone the repository
git clone https://github.com/your-user/ai-code-review-pro.git

# Install dependencies
pip install -r requirements.txt

# Initialize your session
python main.py --api-key YOUR_API_KEY
```

### Configuration

Create a `.env` file to manage your limits and API keys:

```bash
# .env
FREE_TIER_LIMIT=5
STRIPE_SECRET_KEY=sk_test_xxx
```
